Натуральные числа и Арифметика

>> Признаки делимости:
  • На 2: последняя цифра четная.
  • На 3: сумма цифр делится на 3.
  • На 4: две последние цифры образуют число, делящееся на 4.
  • На 5: оканчивается на 0 или 5.
  • На 9: сумма цифр делится на 9.
  • На 10, 100...: оканчивается на соотв. кол-во нулей.

🟡 SECTOR [7–8]: Алгебра / Геометрия (Начала)

Свойства степеней и ФСУ

[ Степени ]
  • aⁿ · aᵐ = aⁿ⁺ᵐ
  • aⁿ / aᵐ = aⁿ⁻ᵐ
  • (aⁿ)ᵐ = aⁿᵐ
  • (ab)ⁿ = aⁿ bⁿ
  • (a/b)ⁿ = aⁿ / bⁿ
  • a⁻ⁿ = 1 / aⁿ
[ ФСУ: Формулы сокращенного умножения ]
  • (a + b)² = a² + 2ab + b²
  • (a - b)² = a² - 2ab + b²
  • a² - b² = (a - b)(a + b)
  • (a + b)³ = a³ + 3a²b + 3ab² + b³
  • (a - b)³ = a³ - 3a²b + 3ab² - b³
  • a³ + b³ = (a + b)(a² - ab + b²)
  • a³ - b³ = (a - b)(a² + ab + b²)

🔵 SECTOR [10]: Тригонометрия / Логарифмы

Тригонометрический атлас

Тождества:
  • sin²x + cos²x = 1
  • 1 + tg²x = 1 / cos²x
  • 1 + ctg²x = 1 / sin²x
Двойной угол:
  • sin 2x = 2 sin x cos x
  • cos 2x = cos²x - sin²x = 2cos²x - 1 = 1 - 2sin²x
  • tg 2x = 2tg x / (1 - tg²x)
Понижение степени:
  • sin²x = (1 - cos 2x) / 2
  • cos²x = (1 + cos 2x) / 2
Сумма и разность:
sin x ± sin y = 2 sin((x±y)/2) cos((x∓y)/2)
cos x + cos y = 2 cos((x+y)/2) cos((x-y)/2)
cos x - cos y = -2 sin((x+y)/2) sin((x-y)/2)

Логарифмы и экспоненты
